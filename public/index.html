What this is

                  
A local Node.js dashboard for HomeEye / Shix / V720-style cameras, built to control live preview, recordings, setup, and recovery from one shell.

                

              

              

                
This project bridges the camera's native protocol with a browser UI so you can inspect live state, apply settings, browse recordings, and handle camera-busy retries without leaving the dashboard.

                
GitHub repository placeholder

The dashboard focuses on camera operations that matter day to day.

Supported cameras

                
This dashboard targets the camera family exposed by the bundled HomeEye / Shix IPC protocol and the decompiled app assets.

              

            

            

              

                HomeEye / Shix IPC
                Primary protocol family used by this project.
              

              

                V720-style cameras
                Devices using the same native SD, preview, and control behavior.
              

              

                JX20130716-class devices
                Compatible native camera stack as bundled in the current bridge.
